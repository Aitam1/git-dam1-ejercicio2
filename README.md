# Programación en Java 🖥️

## Introducción

En el módulo de **Programación** hemos aprendido los fundamentos del desarrollo de software utilizando el lenguaje *Java*. A lo largo del curso hemos trabajado con distintos conceptos esenciales para cualquier programador.

---

## Conceptos principales

### 1. Variables y tipos de datos

En Java existen diferentes tipos de datos:

- **Enteros:** `int`, `long`
- **Decimales:** `double`, `float`
- **Texto:** `String`
- **Booleanos:** `boolean`

### 2. Estructuras de control

Las estructuras de control nos permiten tomar decisiones en el programa:

1. Condicionales (`if`, `else`, `switch`)
2. Bucles (`for`, `while`, `do-while`)
3. Control de flujo (`break`, `continue`, `return`)

### 3. Colecciones

> "Una LinkedList es una estructura de datos dinámica que permite almacenar elementos sin necesidad de definir un tamaño fijo."

Hemos trabajado con:

- Arrays simples
- *LinkedList* de la librería `java.util`
- *Iterator* para recorrer colecciones

---

## Descripción del ejercicio

Programa que gestiona una lista de números introducidos por el usuario:

1. Contar números positivos, negativos y ceros
2. Encontrar el mayor y el menor número
3. Mostrar cuántas veces se repite cada número
    - Usando *Iterator*
    - Con *LinkedList*
4. Eliminar un número indicado por el usuario:
    - Recorriendo la lista con *Iterator*
    - Mostrando la lista final
